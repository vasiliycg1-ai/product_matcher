# Товарный матчер

Программа для подбора товаров из ассортимента под суммы из реестра.

## Возможности
- Подбор товаров целыми штуками
- Корректировка цены для самого дорогого товара (до 2%)
- Сохранение всех исходных колонок
- Логирование в файл

## Установка
```bash
pip install pandas openpyxl

## Использование
Подготовь файлы:

ассортимент.xlsx с колонками: Бренд, Артикул, Кол-во, Цена за 1 шт

реестр.xlsx с колонками: Пояснение, Сумма

Запусти скрипт

Результаты в папке подобранные_товары

## Структура
vt2.py - основной скрипт

output/ - результаты подбора

logs/ - логи работы
